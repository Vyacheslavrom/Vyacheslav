Описание макета html страницы

Данный макет html страницы создан на основе табличной верстки. Макет состоит из пяти основных разделов: заголовка страницы (header), раздела главного меню (main-menu), основной области (main), нижней части (sub-footer) и нижнего колонтитула (footer). Макет является статичным, поскольку размер основной области не зависит от размера окна браузера. Динамически меняет свой размер только область с текстом выбранного раздела. Основная область (main) также поделена на три части: левая колонка (left-side-bar), правая колонка (right-side-bar) и главная колонка (main-bar). Таким образом, перед вами шаблон html страницы с тремя колонками с фиксированным размером. Чтобы растянуть размер страницы на всю ширину экрана достаточно убрать атрибут width из корневой таблицы, для которой определен класс frame.

На самом деле, этот макет не является результатом чисто табличной верстки, поскольку для создания разделов слева и справа основной области используются элементы DIV верстки. Все основные настройки стилей вынесены во внедренные в xhtml документ каскадные таблицы стилей CSS. Познакомиться с деталями html разметки и CSS настроек можно в окне любого браузера, открыв исходный код данной страницы. Этот макет является базовым, на основе которого в дальнейшем будут создаваться более сложные варианты шаблонов интернет страниц.
